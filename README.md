# parallel_k_means_clustering
Parallel implementations of approximate algorithms for K-means clustering using Open MP in C
Python modules leverage the K-means implementation for image segmentation using C bindigs.

## Include
Header files for C package

## Src
C code for the K-means clustering implemented using Lloyds' algorithm. 
Farthtest-first algorithm implemented and leveraged by kmeans.c to find the initial K-centers
Implementation of a simple vector class vec.c

## Python Modules
Image segmentation module leveraging Python's I/O libraries for consistency and C package for efficientcy in the computationally intensive portions of the algorithm. 
Outputs segmented image file.
